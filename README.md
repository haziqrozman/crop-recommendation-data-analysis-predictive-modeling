# Crop Recommendation — Data Analysis & Predictive Modeling

---

## About

This is a course project submitted as part of my **Programming for Data Analytic** course during my Bachelor's degree in Information Technology (Hons) in Internet of Things at Universiti Kuala Lumpur (2021 – 2025).

The project performs exploratory data analysis and predictive modeling on a crop recommendation dataset, using soil nutrients and environmental conditions to determine the most suitable crop for a given set of conditions.

## Quick Preview

| What it covers | How |
|---|---|
| Data preparation | Data understanding, cleaning, and type validation |
| Nutrient analysis | Nitrogen, Phosphorus, Potassium levels across 22 crop types |
| Environmental analysis | Temperature, humidity, pH, and rainfall distributions |
| Predictive modeling | Decision Tree, Random Forest, Gradient Boosting classifiers |
| Feature importance | Rainfall (35.11%) and Humidity (18.72%) as top predictors |

Built with **Python** — pandas, NumPy, seaborn, matplotlib, and scikit-learn.

## Model Results

| Model | Accuracy |
|---|---|
| Decision Tree | 98.64% |
| Random Forest | 99.32% |
| Gradient Boosting | 98.18% |
